### crawlEcommerce
- #### Create virtual environment 
    * ``` python -m venv venv ```
- #### Install package in requirement.txt
    * ``` source venv/Scripts/activate ```
    * ``` pip install -r requirement.txt ```
- #### Config APP in settings.py
    * Config database and parameters for request in settings.py
- #### Run
    * ``` python main.py ```
